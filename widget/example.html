🤖 RAG Chatbot Widget - Integration Guide

Quick Start

Add this single line of code to your website to embed the chatbot widget:

<script src="https://your-domain.com/widget/loader.js" data-api-url="http://localhost:8000" data-position="bottom-right" data-primary-color="#1f77b4"></script>

Configuration Options

You can customize the widget using data attributes:

Attribute Description Default Example
data-api-url API endpoint URL http://localhost:8000 https://api.example.com
data-widget-url Widget JavaScript file URL (optional) Auto-detected https://cdn.example.com/widget.js
data-subject Filter by subject (optional) null (all subjects) Mathematics
data-model-key AI model to use (optional) null (default model) gemini
data-position Widget position bottom-right bottom-left, top-right, top-left
data-primary-color Primary color (hex) #1f77b4 #FF5733
data-text-color Text color (hex) #333333 #000000
data-background-color Background color (hex) #ffffff #F5F5F5
data-button-text Chat button text 🤖
data-welcome-message Welcome message Xin chào! Tôi có thể giúp gì cho bạn? Hello! How can I help?
data-placeholder Input placeholder text Nhập câu hỏi của bạn... Type your question...

Full Example

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Welcome to My Website</h1> <p>This is my website content...</p> <!-- Chatbot Widget --> <script src="https://your-domain.com/widget/loader.js" data-api-url="https://api.example.com" data-subject="Mathematics" data-position="bottom-right" data-primary-color="#1f77b4" data-button-text="💬 Hỏi đáp" data-welcome-message="Xin chào! Tôi có thể giúp gì cho bạn về môn Toán?" data-placeholder="Nhập câu hỏi về Toán học..."> </script> </body> </html>
⚠️ Important Notes:
  • Make sure your API server has CORS enabled for your domain
  • Update data-api-url to point to your actual API endpoint
  • For production, use HTTPS URLs
  • The widget is mobile-responsive and will adapt to smaller screens

Manual Initialization (Advanced)

If you need more control, you can manually initialize the widget:

<script src="https://your-domain.com/widget/chatbot-widget.js"></script> <script> // Wait for widget to load window.addEventListener('load', function() { const widget = new ChatbotWidget({ apiUrl: 'https://api.example.com', subject: 'Mathematics', position: 'bottom-right', primaryColor: '#1f77b4' }); // Access widget instance window.myChatbot = widget; }); </script>
💡 Try it now! The chatbot widget should appear in the bottom-right corner of this page.
